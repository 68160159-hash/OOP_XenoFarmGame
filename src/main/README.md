🚀 XENO-FARM: GALAXY SANCTUARY
Developed by Patcharada

เกมจำลองการเลี้ยงสัตว์ต่างดาวในสไตล์ Retro Pixel Art พัฒนาด้วยภาษา Java (Swing/AWT) 🌌 ภายใต้แนวคิด Model-View-Controller (MVC) เพื่อการจัดการโค้ดที่มีประสิทธิภาพ

🛠️ Technical Architecture (OOP Concepts)
Encapsulation: แยกส่วนข้อมูล (Model) ออกจากส่วนแสดงผล (View) อย่างชัดเจน

Resource Management: คลาส ResourceManager ส่วนกลางที่จัดการโหลด Assets แบบ Static เพื่อประหยัด Memory

Game Loop: ระบบการ Render แบบ 60 FPS โดยใช้ javax.swing.Timer เพื่อให้การเคลื่อนที่ของสัตว์ดูลื่นไหล

Collision-Free AI: สัตว์เลี้ยงมีระบบสุ่มเป้าหมาย (Wander AI) และการคำนวณทิศทางหันหน้า (Facing Direction) แบบไดนามิก

🌟 Key Features
Animated Menu: ระบบหน้าจอหลักพร้อมแอนิเมชันนักบินอวกาศ (Astro) และเอฟเฟกต์ Floating

Inventory System: ระบบจัดการทรัพยากรแยกสายพันธุ์ (Solar Energy & Dew Drop) พร้อมระบบจำกัดความจุ (Cap Limit)

Evolution System: สัตว์เลี้ยงมีการเติบโตและเปลี่ยนรูปร่าง (Sprite Change) เมื่อถึง Level 6

Dynamic Sound: ระบบจัดการเสียง BGM (Loop) และ SFX (One-shot) พร้อมระบบ Mute

Economic Loop: ระบบการข้ามวัน (Next Day) เพื่อรับผลผลิตเป็น Golds จากสัตว์ที่โตเต็มวัย

🎮 How to Play
Login: ระบุตัวตนนักเดินทางเพื่อเริ่มต้นภารกิจ

Selection: เลือกคู่หูตัวแรกระหว่าง Magma Cow (ธาตุไฟ) หรือ Aqua Pig (ธาตุน้ำ)

Care: เลือกให้อาหารที่ตรงตามประเภท (Solar Food ☀️ / Dew Drop 💧)

Grow: พัฒนาฟาร์มผ่านระบบ Next Day เพื่อรับเงินและสุ่มทรัพยากร

Goal: เลี้ยงสัตว์เลี้ยงให้ถึง Level 15 เพื่อพิชิตเกมและก้าวสู่การเป็นสุดยอด Xeno-Farmer! 🏆

⚠️ Hazard Report (Space Predator)
Risk Level: 8% Chance per day
